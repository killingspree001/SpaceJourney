# Voyage - 3D Space Journey

An immersive, cinematic 3D space journey built with **Three.js**, **GSAP**, and **Vite**. Experience a scroll-driven voyage from Earth to the edge of the observable universe.

## 🚀 Features

- **True 3D Environment:** Fly through a field of 25,000 procedurally generated stars.
- **Cinematic Scroll Animations:** Smooth camera transitions and planet reveals powered by GSAP ScrollTrigger.
- **Procedural Planets:** High-quality planet textures and rings generated dynamically on-the-fly.
- **Glassmorphism UI:** Premium frosted-glass overlays for mission data and planet statistics.
- **Mobile Optimized:** Fully responsive experience with adaptive 3D rendering for smooth performance on any device.
- **Post-Processing:** Cinematic bloom and glow effects for a high-end visual feel.

## 🛠️ Tech Stack

- **Three.js:** WebGL 3D rendering engine.
- **GSAP:** Industry-standard animation library.
- **Vite:** Next-generation frontend tooling.
- **CSS3:** Advanced styling with glassmorphism and modern typography.

## 🎮 How to Run

1. **Clone the repository:**
   ```bash
   git clone https://github.com/killingspree001/SpaceJourney.git
   ```

2. **Navigate to the project folder:**
   ```bash
   cd SpaceJourney
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```

5. **Open in browser:**
   Navigate to the local URL provided in your terminal (usually `http://localhost:5173/`).
