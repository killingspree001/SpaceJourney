# Space Journey — A Cinematic 3D Cosmos

An immersive, scroll-driven exploratory experience through our solar system and beyond. Built with **Three.js** and **GSAP**, this project transforms scrolling into a cinematic flight through space.

🚀 **[View Live Demo](https://killingspree001.github.io/SpaceJourney/)**

## ✨ Features

- **Cinematic Camera Flight**: Smooth, scroll-triggered pathing that guides you from Earth to the edge of the observable universe.
- **Procedural 3D Worlds**: Every planet is a real 3D sphere with unique procedural textures, lighting, and rotation.
- **Atmospheric Effects**: High-quality Bloom post-processing and star-field depth for a premium "Interstellar" feel.
- **Dynamic HUD**: Real-time distance tracking and navigational stats as you travel through the void.
- **Glassmorphism Panels**: Modern, readable content cards that float seamlessly over the 3D scene.
- **Fully Responsive**: Optimized for every screen size, from desktop ultra-wides to mobile phones.

## 🛠️ Tech Stack

- **Three.js**: WebGL-powered 3D rendering.
- **GSAP (ScrollTrigger)**: Orchestrating the cinematic camera movement.
- **Vite**: Modern frontend tooling for lightning-fast development.
- **Vanilla CSS**: Custom design system with glassmorphism and modern typography.

## 🏃 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v16+)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/killingspree001/SpaceJourney.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
